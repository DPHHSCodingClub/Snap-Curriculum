Guessing Game

Let's build a number guessing game where the player tries to guess a secret number!

In the last project, Keeping Score, you created global variables such as score that kept track of how many times players successfully clicked Alonzo. We call them global because all parts of the code can access the value stored in them.

However, it would be a pretty boring guessing game if anyone could access what the secret number is. Instead, we can use local variables (which Snap! calls scripting variables) that only parts of the code have access to.

If you've done the Row of Dollhouses project, then you've already seen scripting variables: The inputs to custom blocks only exist within the block and can't be used outide of them.

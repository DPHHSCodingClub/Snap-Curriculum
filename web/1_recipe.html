Make a Recipe (HTML/CSS)

HTML, CSS, JavaScript are coding languages used to create the websites and apps that you use every day. In this lab, we'll learn what HTML is and how to use it. You don't need to have any previous knowledge!

  1. What is the "web"? Click here to watch a short video about the web and HTML.

  2. Below is the code for a very simple HTML website. The code is made of tags (things in angle-brackets, <...>) that mark up your webpage contents. There's usually a head and a body. Many HTML tags come in pairs, called opening and closing tags. Click here to watch a short video explanation. 

    <!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>My Cool Webpage</title>
  </head>

  <body>
    <p>Hello world!</p>
  </body>
</html>

  3. Now, you'll make your own website. Create a Replit account.

  4. Visit bit.ly/MakeARecipe to open a starter project. Press the blue "Fork" button to make your own copy.

  5. Think of a recipe that you want to share with others. It doesn't have to be real food, it can be a joke recipe too! If you can't think of one, try googling a recipe for your favorite dish online and use that for the rest of the project.

    Let's make a webpage out of it! Open the index.html file on your repl.it project. We will use five main kinds of HTML tags, shown below.
    Let's start simple, add the line <h1>Ham and Cheese Sandwich</h1> (or put the name of your recipe instead) between the lines that say <body> and </body>. Click the button on the top and see what that did to your website.

    Try removing the </h1> and rerun your website. What happened to the title?

  6. Recipes today always start with an introduction, talking about the dish itself or explaining its history or its importance to the author. With a new line under your <h1>...</h1> heading tag, add a <p>...</p> paragraph tag with your introduction!

  7. When you're done, the webpage should look similar to the below image.

  8. All the information is there, but it's a bit plain. Let's spice it up! Open the style.css file on your repl.it project. Add the code: 

    body {
  color: lightgreen;
}

    What happened? Your website browser (Chrome, Safari) looks for all HTML body tags and colored them light green! Notice that HTML and CSS are two different languages, so we're using curly braces { and } instead of brackets < and >.
